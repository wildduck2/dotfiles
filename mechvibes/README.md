# Mechvibes

Mechanical keyboard sound simulator config, managed via GNU Stow.

## Requirements

- **[Mechvibes](https://mechvibes.com/)** (mechvibes-bin from AUR)

## Setup

```bash
cd ~/dotfiles

# Install dependencies and stow config
./mechvibes/setup.sh

# Or via the main setup
./setup.sh mechvibes
```

## Configuration

- **Sound pack**: NK Cream (`default-nk-cream`)
- **Volume**: 50%

Config is stored at `~/.config/Mechvibes/config.json`.
